"# Distributed Database & Processing System" 
"" 
"## Overview" 
"A distributed database and processing system inspired by Hadoop, Spark, and MapReduce architectures. Built with **Go** as the orchestration language and **MySQL** as the underlying storage engine." 
"" 
"## Architecture" 
"" 
"- **Master Node (Go)** - API Gateway, Coordinator, Metadata Manager, Request Router" 
"- **Worker Nodes** - Go & Python workers with local MySQL storage (sharded data)" 
"- **Reducer Node** - Aggregates partial results from workers" 
"- **Metadata Database (MySQL)** - Stores cluster configuration, shard mappings, worker status" 
"" 
"## Features" 
"" 
"? Distributed sharded storage" 
"? Distributed query execution (parallel across workers)" 
"? MapReduce workflow (word count example)" 
"? Reducer-based aggregation" 
"? Health monitoring & heartbeats" 
"? API key authentication" 
"? Leader election for master failover" 
"? Heterogeneous workers (Go + Python)" 
"? Web UI Dashboard" 
"? RESTful API" 
"" 
"## Tech Stack" 
"" 
"- **Go** - Master, Reducer, Go Worker" 
"- **Python** - Python Worker (Flask)" 
"- **MySQL/MariaDB** - Metadata & worker storage" 
"- **HTML/CSS/HTMX** - Web UI" 
"" 
"## Prerequisites" 
"" 
"- Go 1.21+" 
"- Python 3.8+" 
"- MySQL/MariaDB (XAMPP recommended)" 
"" 
"## Quick Start" 
"" 
"### 1. Clone the repository" 
"\`\`\`bash" 
"git clone https://github.com/your-username/distributed-database-system.git" 
"cd distributed-database-system" 
"\`\`\`" 
"" 
"### 2. Setup MySQL" 
"\`\`\`sql" 
"CREATE DATABASE distributed_metadata;" 
"CREATE DATABASE worker1_db;" 
"CREATE DATABASE worker2_db;" 
"\`\`\`" 
"" 
"### 3. Install dependencies" 
"\`\`\`bash" 
"# Master & Reducer" 
"cd master && go mod init master && go get github.com/go-sql-driver/mysql github.com/gorilla/mux gopkg.in/yaml.v3" 
"" 
"# Go Worker" 
"cd ../worker-go && go mod init worker-go && go get github.com/go-sql-driver/mysql" 
"" 
"# Python Worker" 
"cd ../worker-py && pip install flask mysql-connector-python pyyaml requests" 
"\`\`\`" 
"" 
"### 4. Start the system" 
"\`\`\`bash" 
"# Terminal 1: Reducer" 
"cd reducer && go run main.go" 
"" 
"# Terminal 2: Go Worker" 
"cd worker-go && go run main.go" 
"" 
"# Terminal 3: Python Worker" 
"cd worker-py && python app.py" 
"" 
"# Terminal 4: Master" 
"cd master && go run main.go" 
"\`\`\`" 
"" 
"### 5. Access the UI" 
"Open browser: \`http://localhost:8080/ui\`" 
"" 
"## API Examples" 
"" 
"\`\`\`bash" 
"# Create database" 
"curl -X POST http://localhost:8080/v1/db/create -H \"X-API-Key: your-key\" -d '{\"name\":\"testdb\"}'" 
"" 
"# Create table" 
"curl -X POST http://localhost:8080/v1/table/create -H \"X-API-Key: your-key\" -d '{\"db_name\":\"testdb\",\"table_name\":\"users\",\"schema\":\"id INT PRIMARY KEY, name VARCHAR(100)\",\"shard_key\":\"id\"}'" 
"" 
"# Insert data" 
"curl -X POST http://localhost:8080/v1/insert -H \"X-API-Key: your-key\" -d '{\"db_name\":\"testdb\",\"table_name\":\"users\",\"rows\":[{\"id\":1,\"name\":\"Alice\"}]}'" 
"" 
"# Distributed query" 
"curl -X POST http://localhost:8080/v1/select -H \"X-API-Key: your-key\" -d '{\"db_name\":\"testdb\",\"table_name\":\"users\",\"query\":\"SELECT COUNT(*) FROM users\"}'" 
"\`\`\`"
